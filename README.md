# Todo App Backend

Express.js REST API for the Advanced Todo App.

##  Links

- **Frontend Repo**: [https://github.com/Sub-47/Advanced-To-do](https://github.com/Sub-47/Advanced-To-do)
- **Live API**: [https://todo-backend-fwik.onrender.com](https://todo-backend-fwik.onrender.com)

## Tech Stack

- Node.js
- Express.js
- PostgreSQL
- Render (deployment)

## API Endpoints

- `GET /todos` - Get all todos
- `POST /todos` - Create a new todo
- `PUT /todos/:id` - Update a todo
- `DELETE /todos/:id` - Delete a todo

##  Setup
```bash
npm install
npm start
```

## Deployment

Deployed on Render. Auto-deploys on push to main branch.
